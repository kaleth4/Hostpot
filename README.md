# Hostpot

Script en bat para habilitar una zona de cobertura inalambrica para movil de manera rapida solo con 1 click o cuando no esta la opcion en la laptop de zona de cobertura movil HOTSPOT.
Metodo #1
1.Abre configuracion
2.Entra en red e internet
3.Zona de cobertura inalambrica movil
4.Compartir mi conexion a internet desde ...
5.Activar Compartir con otros dispositivos
6.Escanear el QR

Metodo #2
1.Descarga el BAT
2.Edita el nombre de la red y la contraseña
3.Guarda y ejecuta como administrador
4.Activa en el movil
